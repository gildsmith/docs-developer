# Packages

Gildsmith is composed of focused packages that are developed in this monorepo and
split to their own repositories.

| Package | Purpose |
| --- | --- |
| `gildsmith/anvil` | Package workbench and tooling foundation. |
| `gildsmith/contract` | Shared interfaces for commerce concepts. |
| `gildsmith/product` | Product catalog models, controllers, factories, and migrations. |
| `gildsmith/skeleton` | Starter template for new first-party packages. |
| `gildsmith/support` | Shared support code for package implementations. |
| `gildsmith/docs` | VitePress documentation site. |
